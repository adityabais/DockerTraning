Link: https://docs.docker.com/compose/install/


Run following commands to install Docker-Compose
1) sudo curl -L "https://github.com/docker/compose/releases/download/1.24.1/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose

2) sudo chmod +x /usr/local/bin/docker-compose

Run following command to check if installation was successful or not.

3) sudo docker-compose --version

$ docker-compose --version
docker-compose version 1.24.1, build 1110ad01
